WHAT IS THIS?
-------------
TaskFlow is a web application that helps teams manage their projects and tasks.
Think of it like a digital to-do list, but for teams!

It's completely FREE, requires NO credit card, and you can use it forever.


MAIN FEATURES
-------------
1. Create Projects
   - Organize your work into different projects
   - Example: "Website Redesign", "Marketing Campaign", etc.

2. Manage Tasks
   - Create tasks within projects
   - Assign tasks to team members
   - Set due dates
   - Track progress (To Do → In Progress → Done)

3. Team Collaboration
   - Add team members to projects
   - Everyone sees their assigned work
   - Real-time updates

4. Two User Roles
   - ADMIN: Can create/delete everything, manage team
   - MEMBER: Can view assigned work and update task status

5. Beautiful Design
   - Modern, professional interface
   - Works on phones, tablets, and computers
   - Smooth animations and easy to use

HOW IT WORKS
------------
1. SIGN UP
   - Go to the website
   - Click "Get Started Free"
   - Enter email and password
   - Account created instantly!

2. CREATE A PROJECT
   - Click "New Project"
   - Give it a name (e.g., "Website Redesign")
   - Add a description (optional)
   - Done!

3. ADD TEAM MEMBERS
   - Click the user icon on your project
   - Select team members from the list
   - They can now see the project

4. CREATE TASKS
   - Click "New Task"
   - Enter task details
   - Assign to a team member
   - Set a due date
   - Save!

5. TRACK PROGRESS
   - View all tasks in the dashboard
   - Update status as work progresses
   - See statistics (completed, overdue, etc.)
   - Filter by project or status
